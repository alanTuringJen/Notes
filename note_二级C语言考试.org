* 基础知识
- 程序执行的效率与数据的存储结构密切相关
- 队列结构：先到先服务
- 循环队列的存储空间为 Q(1:35)，初始状态 front = rear = 35. 经过一系列入队与退队的运算之后，front = 15, rear = 15, 现在，循环队列中的元素个数是【0 或 35】。
- 二叉树中，度为0的结点数等于度为二的结点数加一，即 n2 = n0 - 1，叶子结点即度为0的结点。
- 快速排序最坏的情况：n(n-1)/2。
- 多态：同一个操作可以是不同对象的行为。
- 构成计算机软件的是：程序、数据和文档。
- 软件设计中模块划分应该遵循的准则：高内聚低耦合
- 黑盒测试：边界值分析、等价类划分、错误推倒
- 软件测试：单元测试、集成测试、确认测试、系统测试
- 软件需求分析：需求获取、需求分析、编写需求规格说明书、需求评审
- 结构化方法需求分析工具：数据流图、数据字典、判定树、判定表
- 数据字典定义的对象都包含于【数据流图】
- 需求规格说明书不需要说明【软件可行性研究的依据】
- 关系模型中，每个二维表称为一个关系
- 数据库设计过程不包括【算法设计】
- 自然连接时，要求含有一个或多个【属性】
* C语言概述
- 每一个后缀为 .c 的C语言源程序都可以单独进行编译
- C 编译程序把文件后缀为 .c 的源文件编译成文件后缀为 .obj 的二进制文件

** 数据类型
- 不同C语言规定标识符的有效长度可能会不同，但是没有限制最大长度
- .914 合法
- "Hello"这个字符串常量有6个元素，不要忘记 '\0'
- 不能用字符串常量对字符数组名进行整体赋值操作
- 两个连续的单引号('')不是合法的字符常量，两个连续的双引号("")是合法的字符串常量，因为有 '\0'。

* 运算符和表达式
- sizeof(double) 是一个整形表达式
- sizeof是C语言中的一个操作符，不是函数调用
- 逗号表达式的值为其中最后一个表达式的值
- static定义的变量，在函数运行结束之后，变量不会消失
- || 与 && 执行一般就可以知道结果之后，后面的不再执行
- 赋值表达式左边必须是一个常量
- 位运算的对象只能是整形或者字符型数据
* 基本语句
- printf 和 scanf 函数中都可以指定数据的宽度
- printf中，用 "%%" 打印字符 "%"
